# 📊 Data Analysis and Visualization with Python

## 📌 Overview
This project demonstrates basic **data analysis and data visualization techniques using Python**.  
It uses libraries like **Pandas** and **Matplotlib** to explore datasets, summarize information, and create meaningful visualizations such as bar charts, line plots, scatter plots, and multi-line graphs.

---

## 🧰 Technologies Used
- Python 🐍
- Pandas 📊
- Matplotlib 📈

---

## 📁 Dataset
The project uses a CSV dataset:  
`woocommerce-product-export.csv`

It contains monthly product sales data including:
- Month number
- Total profit
- Sales of different products (facecream, facewash, toothpaste, bathingsoap, shampoo, moisturizer)

---

## 📊 Features & Tasks Completed

### 1. Data Loading & Exploration
- Read CSV file using `pandas`
- Display dataset information using `info()`
- Show statistical summary using `describe()`
- Display first and last 5 rows using `head()` and `tail()`

---

### 2. Data Selection
- Extract specific columns:
  - `month_number`
  - `total_profit`

---

### 3. Data Visualization
